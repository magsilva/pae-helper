Instruções para envio de programas em PHP

  1. Todos os arquivos relativos ao trabalho devem estar compactados em um único arquivo .ZIP.
  2. Os seguintes arquivos sempre devem estar presentes:
    • Arquivos com código-fonte (.py).
    • Um arquivo texto com o nome LEIAME.TXT contendo instruções para executar o programa.
  3. Arquivos diferentes dos citados acima (e não referenciados no LEIAME.TXT) serão ignorados ou deletados.
  4. O único requisito para compilar e executar o programa deve ser o Python 2.2.x. Em outras palavras, deverá ser possível executar o programa usando o comando "python", preferencialmente sem parâmetros (a menos que o exercício exija o contrário).
  5. Exercícios sem implementação deverão estar também dentro de um .py. Coloque a resposta dentro de um comentário.
  6. Envie o arquivo .zip para o email do estagiário (magsilva@icmc.usp.br) ou do professor da disciplina (renata@icmc.usp.br). Coloque, no corpo da mensagem, o nome dos alunos e respectivos números USP.

Instruções para envio de programas em Java

  1. Todos os arquivos relativos ao trabalho devem estar compactados em um único arquivo .ZIP.
  2. Os seguintes arquivos sempre devem estar presentes:
    • Arquivos com código-fonte (.java).
    • Um arquivo texto com o nome LEIAME.TXT contendo instruções para executar o programa.
  3. Arquivos diferentes dos citados acima (e não referenciados no LEIAME.TXT) serão ignorados ou deletados.
  4. O único requisito para compilar e executar o programa deve ser uma SDK 1.4.x. Em outras palavras, deverá ser possível gerar o programa por meio do javac e executar com o java, preferencialmente sem nenhum parâmetro além do arquivo (a menos que o exercício exija o contrário).
  5. O classpath do ambiente em que será testada a aplicação incluirá os pacotes Java e o diretório local.
  6. Exercícios sem implementação deverão estar também dentro de um .java. Coloque a resposta dentro de um comentário.
  7. Envie o arquivo .zip para o email do estagiário (magsilva@icmc.usp.br) ou do professor da disciplina (renata@icmc.usp.br). Coloque, no corpo da mensagem, o nome dos alunos e respectivos números USP.
